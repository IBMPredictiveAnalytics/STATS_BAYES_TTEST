Bayesian T Test

The procedure calculates the Bayes factors and the posterior probabilities for a t test. The t test can be a one-sample test against a constant, an unpaired test, or a paired test. The posterior distributions for a single prior scale factor can also be displayed.

The Bayes factor is the relative evidence in the data for one model over another. See the reference below for an introductory discussion.

Split files and the Statistics case weights are not supported by this procedure. Cases with missing values will be omitted from the analysis.

Variables to Test Specify either one or two test variables. For an unpaired test, specify one variable and a group variable or one variable and a test value. For a paired test specify two variables.

Group Specify a group identification variable if doing an unpaired test with two groups. Otherwise leave this field empty. The group variable must have exactly two distinct non-missing values.

Paired test Check this box to do a paired test. In that case there must be two variables in the test field and no group variable.

Test Value For a one-sample test against a specific value, enter the value here and leave the group field empty.

Options

Display posterior distribution Check this box to display the posterior distribution for the first prior scale value listed.

Prior Scale Values Choose the scales for the prior on all the standardized slopes. Multiple values can be entered separated by a blank. The scale parameter controls the expected true standardized effect size. For a particular scale 50% of the true effect sizes are within the interval. Values corresponding to the Bayesian regression procedure are medium: (-.7071, .7071); wide: (-1, 1), and ultrawide: (-1.414, 1.414). One or more of these keywords or numbers can be entered.

Number of samples Specify the number of iterations (samples) to be drawn in calculating the posterior distribution.

Workspace Action If Retain is selected, the workspace containing the results is retained and can be used in an R program in the current session. Otherwise, the workspace is cleared.

Save model file Enter a file specification to save the workspace results to an R file for later use.

Additional Features

This dialog generates syntax for the STATS BAYES TTEST extension command. To display syntax help for this command, run the following syntax:

STATS BAYES TTEST /HELP.

In Statistics version 23 or later, you can place the cursor within the command in the Syntax Editor and press F1.
