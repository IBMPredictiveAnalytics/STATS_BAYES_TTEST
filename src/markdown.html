STATS BAYES TTEST Extension Command

Calculate Bayesian T Test

STATS BAYES TTEST VARIABLES = one or two variables to test*
GROUP = group variable
TESTVALUE = test value
PAIRED = NO** or YES

/OPTIONS POSTERIOR = YES** or NO
ITERATIONS = number
PRIORSCALE = "one or more prior parameter values"

/SAVE WORKSPACE = CLEAR** or RETAIN
MODELFILE = "file specification for results"

* Required
** Default

STATS BAYES TTEST /HELP prints this information and does nothing else.

STATS BAYES TTEST salary
GROUP = gender
/OPTIONS PRIORSCALE = ".7071 1.414".

Split files and weights are not honored by this procedure. Cases with missing values are omitted.

This procedure calculates the Bayes factors and the posterior probabilities for a t test. The t test can be a one-sample test against a constant, an unpaired test, or a paired test. The posterior distributions for a single prior scale value can also be displayed.

The Bayes factor is the relative evidence in the data for one model over another. See the reference below for an introductory discussion.

VARIABLES Specify either one or two test variables. For an unpaired test, specify one variable and a group variable or one variable and a test value. For a paired test specify two variables and no group variable. The test variables must have a scale (continuous) measurement level.

GROUP Specify a group identification variable if doing an unpaired test with two groups of cases. Otherwise omit this keyword. The group variable must have exactly two distinct, non-missing values.

TESTVALUE For a one-sample test against a specific value, specify the value here, and do not specify a group variable.

OPTIONS

POSTERIOR specifies whether the posterior distribution of the test is displayed. The posterior distribution for the first prior scale value listed is used.

ITERATIONS specifies the number of iterations (samples) to use for the posterior. The default value is 1000.

PRIORSCALE specifies the scales for the prior. Multiple values can be entered. The entire set of values must be enclosed in quotation marks. The default value is .7071 (sqrt(2)/2). The scale parameter controls the expected true standardized effect size. For a particular scale 50% of the true effect sizes are within the interval. Values corresponding to the Bayesian regression procedure are medium: (-.7071, .7071); wide: (-1, 1), and ultrawide: (-1.414, 1.414). One or more of these keywords or other numbers can be entered.

SAVE

WORKSPACE specifies whether the workspace should be cleared or retained after the procedure completes. By default, it is cleared.

MODELFILE specifies a file name to which the model information will be written as an R workspace for later use.
